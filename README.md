# Container Logistics Backend

Backend application developed as an academic project for Backend de Aplicaciones (UTN).
The system manages the logistics of container transportation using a microservices architecture.

__Main Features__

Container transport request management

Route planning with multiple segments and deposits

Cost and delivery time estimation

Assignment of trucks based on capacity and availability

Real-time transport status tracking

Role-based access (Client, Operator, Transporter)

__Architecture__

Microservices-based architecture

API Gateway

RESTful APIs

Docker & Docker Compose

__Security__

Authentication and authorization using Keycloak

JWT-based security

Role-based access control

__Technologies__

Java

Spring Boot

Spring Security

REST APIs

Docker / Docker Compose

Swagger / OpenAPI

SQL (relational database)
